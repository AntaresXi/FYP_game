Finite Automata Game
---------------------

Welcome to the Finite Automata Game! This game is designed to help you learn and explore the concepts of finite state automata and regular expressions.

Instructions:
- Open 'webspace.html' in a web browser to start the game.
- Follow the on-screen instructions to navigate through the game.
- Use the tutorial section to learn about regex and finite state automata history.

Enjoy the game and happy learning!
